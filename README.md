<h1> Use Case </h1>
If you're a Django Developer who's done with doing same things over and over again to start/setup your project , this tool is for you


<h1> Usage </h1>

```
djit [project_name] [list_of_apps]
# for example
djit core users,shop

OR

djit
```

> [!Important]
> djit with no args triggers default behaviour (project name = core and no apps).


> [!TIP]
> list_of_apps format is app1,app2,app3,...   Seperate apps with , and don't use [],(),"",{},...
